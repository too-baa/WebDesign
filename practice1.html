Python Language

Origin of Python

Python was created by Guido Van Rossum and was released in February 1991. Python got its name from a BBC comedy series from 70s- "Monty Python's Flying Circus".

It can be used to follow both, procedural and Object Oriented approach of programming. It is an open source language and free to use.

Python is based on or influenced with two programming languages: ABC(replacement of BASIC) and Modula-3.

Features of Python which make it popular

1. It is an object oriented language.
2. Expressive language.
3. Cross-platform language.
4. Free and open source.
5. Variety of usage/application.

Limitations of Python

1. Not the fastest language.
2. Lesser libraries than C, Java and Perl.
3. Not strong on type-binding.
4. Not easily convertible.
